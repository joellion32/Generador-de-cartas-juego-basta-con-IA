# Code of Conduct

Todos los colaboradores deben respetar un entorno colaborativo y profesional.

Reglas básicas
- Sé respetuoso y profesional.
- Evita lenguaje ofensivo y discriminatorio.
- Reporta comportamiento inadecuado abriendo un issue o contactando a los responsables del proyecto.

Aplicación
- Los mantenedores se reservan el derecho de eliminar contenido o bloquear usuarios que incumplan estas normas.
