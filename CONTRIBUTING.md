# Contributing

Gracias por contribuir a este proyecto. Sigue estos pasos para colaborar de forma efectiva.

1. Fork y branch
- Crea un fork y trabaja en una branch descriptiva: `feat/nueva-funcionalidad`.

2. Estilo de código
- Sigue las reglas de TypeScript y ESLint si están configurados.
- Formatea con `prettier` si está configurado.

3. Commits y PR
- Mensajes claros: `feat:`, `fix:`, `chore:`, `docs:`
- Incluye descripción, pasos para reproducir y screenshots si aplica.

4. Tests
- Añade tests unitarios o de integración cuando agregues lógica crítica.

5. Revisiones
- Espera que tu PR sea revisado; responde comentarios y actualiza la branch.

Contacto
- Abre un issue si algo no encaja o para discutir grandes cambios.
